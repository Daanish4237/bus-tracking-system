# Product: Bus Tracking System

A real-time bus tracking web application for passengers. It lets users view bus routes, select a drop-off stop, and track live GPS locations of buses along their chosen route.

## Core Features

- View all available bus routes and their stop sequences
- Select a drop-off stop on a route
- Real-time bus location updates (polling every 10 seconds)
- Stop-by-stop progress tracking (completed / current / upcoming)
- Responsive UI for desktop and mobile

## Sample Data

Three pre-loaded routes for development/testing:
- Route 101 - Downtown Express (5 stops)
- Route 202 - Uptown Local (6 stops)
- Route 303 - Crosstown Shuttle (5 stops)
